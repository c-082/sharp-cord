# SharpCord
SharpCord is a multipurpose Discord bot that responds to both **text commands** and **slash commands**. It was primarily made for learning purposes, specifically, to learn about asynchronous programming, events, event handlers, and dependency injection.

## Features
- Text commands (e.g., `$ping`)
- Slash commands (e.g., `/ping`)
- Simple and fast responses

## Getting Started
### Prerequisites
- Discord Bot Token
- [.NET 10 Runtime](https://dotnet.microsoft.com/)

### Running the Bot
1. Clone the repository:
```bash
git clone https://github.com/c-082/sharp-cord.git
cd sharp-cord
```

2. Set your bot token in environment variables:
`BOT_TOKEN="your_bot_token_here"`

3. Run the bot:
```bash
dotnet run
```
